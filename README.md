# RBTI — Research Behavior Type Indicator

> MBTI已经过时，SBTI不够学术，RBTI来了。

面向科研人的讽刺人格测试。562种科研人格，10大学科，20道题，2分钟测出你的科研灵魂。

**在线体验**: [rbti.picasso-lab.com](https://rbti.picasso-lab.com)

---

## 这是什么？

一个用 ∫₀^{114514} e^{-x²} dx 来暗示人格总数的科研人格测试。

10大学科全覆盖，每个学科都有专属题目和人格：

| 学科 | 人格数 | 示例 |
|------|--------|------|
| 计算机/信息 | 100 | VIM（进去出不来）、LLM（万金油）、TCP（三遍确认） |
| 生物/医学 | 80 | 条带玄学、支原体恐惧、养细胞的 |
| 机械/电气/自控 | 60 | 仿真永不收敛、通电冒烟、PID炼丹 |
| 物理/化学/材料 | 60 | 量子态、光路调节、产率2% |
| 土木/环境/能源 | 60 | 28天不达标、地基玄学、电池循环 |
| 商科/社科 | 61 | 方差不齐、内生性噩梦、p值战士 |
| 数学/统计 | 60 | 证明强迫症、猜想收藏家、逼近论 |
| 人文/法律 | 52 | 知识产权（论文被抄了）、解构主义 |
| 艺术/建筑 | 15 | 甲方放了红木、渲染出图 |
| 通用 | 9 | 死线战士、冒名顶替、引用狂魔 |

还有6种隐藏人格等你触发。

### 10维度评分体系

| 维度 | 含义 | 学科定制 |
|------|------|---------|
| R1 | 理论↔应用 | 每个学科有专属版本 |
| R2 | 深度↔广度 | 部分学科定制 |
| W1 | 独立↔协作 | 通用 |
| W2 | 按学科变化 | CS: 代码↔数学、Bio: 湿实验↔干实验、Math: 证明↔计算 等 |
| P1 | 质量↔数量 | 通用 |
| P2 | 计划↔即兴 | 通用 |
| M1 | 好奇↔功利 | 通用 |
| M2 | 学术↔工业 | 通用 |
| S1 | 抗压↔脆弱 | 每个学科有专属版本 |
| S2 | 社交↔独处 | 每个学科有专属版本 |
